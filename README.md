# quran-halaqa
Checkin for quran halaqa


## Installation

```console
git clone git@github.com:iamjazzar/quran-halaqa.git
```

```console
mkvirtualenv quranhalaqa
```

```console
pip install -r requirements.txt
```

```console
npm i
```

## Development

```console
python manage.py runserver
```

```console
npm run tailwind-watch
```


## Deployment

```
npm run tailwind-build
```
